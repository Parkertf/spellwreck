# Spellwreck

A party-style spelling bee game where one person stands with their back to the screen while friends see the word they have to spell.

## How to Play

1. Add players and pick your settings (difficulty, timer)
2. Hit **Start Game**
3. The speller stands with their back to the TV/screen
4. The audience reads the word aloud
5. The speller tries to spell it
6. Someone taps **Correct**, **Wrong**, or **Skip**

## Features

- 100+ commonly misspelled words (Canadian spelling)
- Difficulty tiers: Easy, Medium, Hard
- Configurable countdown timer with play/pause
- Sound effects for correct, wrong, skip, and time's up
- Streak tracking with fire badges
- Scoreboard with crown/clown emojis
- End-of-game summary with podium, best streaks, and most-missed words
- Keyboard shortcuts: `1`/`←` Correct, `2`/`↓` Wrong, `3`/`→` Skip
- All state persists in localStorage

## Running

Open `index.html` in a browser. No build step, no dependencies, no server needed.
